# 我的工具集

个人工具导航网站，包含多个实用工具和演示项目。

## 工具分类

### 开发者工具 (`tools/`)
- **多功能工具集** - Base64编解码、时间戳转换、URL编解码等
- **ULID 生成器** - 时间排序的唯一标识符生成
- **TSID 生成器** - 时间排序ID生成
- **MyBatis 格式化** - SQL日志格式化工具
- **API 测试工具** - HTTP请求测试

### 地图与可视化 (`botw/` / `hdmap/`)
- **旷野之息地图** - 塞尔达传说：旷野之息互动地图
- **高精度地图** - 路网监测高清地图平台

### 学习资源 (`study/opencv/`)
- OpenCV 图片/视频操作基础/进阶/高级教程

### 交互演示 (`sandbox/`)
- 3D 城市模型、拖拽交互等 Cesium.js 实验

### AI 工具 (`aitools/`)
- AI 工具面板

## 技术栈

纯原生 HTML/CSS/JS，无构建系统。

## 部署

直接部署静态文件到任意静态托管服务。
